# Studium Projekt

### Crossplattform-Chat mit C# und Xamarin:
Dieses Projekt diente als Projekt zum Abschluss eines Moduls in meinem Informatik Studium und wurde von mir und einem Kommilitonen erstellt.
Es bietet eine einfachklassische Chat Funktion, sowohl als Android und Windows App, als auch als Web-App.
